Chapter summaries: Attributes All HTML elements can have attributes The title attribute provides additional "tool-tip" information The href attribute provides address information for links The width and height attributes provide size information for images The alt attribute provides text for screen readers At W3Schools we always use lowercase attribute names At W3Schools we always quote attribute values with double quotes Headings Search engines use the headings to index the structure and content of your web pages. Use HTML headings for headings only. Don't use headings to make text BIG or bold. "head" part is used to show metadata about the document and is not displayed on webpage, Metadata typically define the document title, character set, styles, links, scripts, and other meta information. 	Style
		HTML style attribute has the syntax:  , where property and value are CSS data
			The "background-color" property defines the background color for an HTML element, style="background-color:whatever"
			The "color" property defines the text color for an HTML element, style="color:blue;"
			The "font-family" property defines the font to be used for an HTML element, style="font-family:verdana;"
			The font-size property defines the text size for an HTML element, style="font-size:300%;"
			The text-align property defines the horizontal text alignment for an HTML element, style="text-align:center;"
			How to put multiple styles into one line: style=" color:red; text-align:center; font-family:verdana; "
